# Usage

## Starting with Docker

### Download Image

`docker pull hidetodong315/kuroko:latest`

### Or build a new one
docker build -t [name]/kuroko .

### Start Container

`docker run -dt -p 8050:8050 -v data/config:data/config`

## Starting with NPM

`pnpm run start`

or prefered using `nr`

`npm i -g nr`

then

`nr start`
